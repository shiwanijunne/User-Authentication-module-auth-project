# This is a Flask-based user authentication project.

## Features
- User Registration
- User Login
- Dashboard
- Change Password
- Session-based Authentication

## Technologies Used
- Python
- Flask
- HTML
- MySQL

## How to Run
1. Install dependencies  
   pip install flask mysql-connector-python
2. Run the app  
   python app.py
3. Open browser  
   http://127.0.0.1:5000/
